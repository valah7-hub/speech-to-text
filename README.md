# Speech-to-Text v1.9.0

**Windows-приложение для голосового ввода текста.** Зажмите горячую клавишу — говорите — текст появляется в любом приложении в реальном времени.

Работает полностью локально. Ничего не отправляется в интернет.

---

## Возможности

| Функция | Описание |
|---------|----------|
| **Голосовой ввод** | Зажали хоткей — говорите — текст вставляется |
| **Стриминг** | Текст появляется по мере произнесения, не после |
| **GPU-ускорение** | NVIDIA CUDA — в 5-10 раз быстрее CPU |
| **Русский + English** | Русская речь, английские термины сохраняются латиницей |
| **Голосовые команды** | «Точка», «запятая», «новая строка» — знаки препинания |
| **Автопрослушивание** | Записывает автоматически когда вы говорите (без хоткея) |
| **Транскрибация файлов** | mp3, wav, m4a — текст с таймкодами, экспорт SRT/VTT |
| **Спектрограф** | Индикатор с полосками, реагирующими на голос |
| **Приватность** | 100% локально, модели на вашем компьютере |

## Установка

### Требования

- **Windows 10/11**
- **Python 3.10+** ([скачать](https://python.org/downloads) — при установке отметить "Add to PATH")
- **NVIDIA GPU** (опционально, для ускорения)

### Быстрая установка (рекомендуется)

```bash
git clone https://github.com/valah7-hub/speech-to-text.git
cd speech-to-text
install.bat
```

Скрипт `install.bat` автоматически:
1. Проверит наличие Python
2. Установит все зависимости (~50 MB с PyPI)
3. Установит движок Faster-Whisper (~20 MB с PyPI)
4. Предложит установить GPU-ускорение (~2.5 GB, опционально)

### Запуск

- **Без консольного окна:** дважды кликните `start.vbs`
- **С консолью (для отладки):** `python app.py`

### Что скачивается и откуда

| Что | Откуда | Размер | Когда |
|-----|--------|--------|-------|
| Python-пакеты | PyPI (pypi.org) | ~50 MB | При установке |
| Faster-Whisper | PyPI | ~20 MB | При установке |
| PyTorch + CUDA | pytorch.org | ~2.5 GB | При установке (опционально) |
| Модель Whisper | Hugging Face (huggingface.co) | 75 MB — 3 GB | При первом запуске |

Все источники — крупные стабильные платформы. Адреса не меняются.

### Ручная установка

```bash
pip install -r requirements.txt
pip install faster-whisper
pip install torch --index-url https://download.pytorch.org/whl/cu126  # GPU
python app.py
```

## Использование

### Голосовой ввод

1. Запустите `python app.py`
2. На экране появится маленький овальный индикатор со спектрографом
3. Откройте приложение для ввода (Notepad, Word, браузер, мессенджер)
4. **Зажмите `Ctrl+Space`** (или настроенный хоткей)
5. Говорите — текст начнёт появляться через 2-3 секунды
6. **Отпустите** — финальный текст вставится

### Индикатор

- **Левый клик** — Настройки
- **Правый клик** — Меню (Настройки / История / Файлы / Выход)
- **Перетащить** — Зажать и двигать в любое место

### Цвета спектрографа

| Цвет | Состояние |
|------|-----------|
| Серый | Готов к записи |
| Красный | Идёт запись (полоски прыгают с голосом) |
| Жёлтый | Обработка |
| Зелёный | Текст вставлен |

### Хоткеи

По умолчанию: `Ctrl+Space`. Можно сменить в настройках на:

- Любую комбинацию клавиш (`Ctrl+Shift+R`, `F9`, ...)
- **Кнопку мыши** (средняя, боковые)

### Автопрослушивание (VAD)

Включите в настройках «Автопрослушивание» — приложение будет слушать микрофон и автоматически записывать когда вы говорите. Хоткей не нужен.

## Настройки

Левый клик по индикатору — Настройки.

### Распознавание

| Параметр | Варианты | Описание |
|----------|----------|----------|
| Движок | faster-whisper, whisper | faster-whisper в 2-4x быстрее |
| GPU | вкл/выкл | NVIDIA CUDA ускорение |
| Модель | tiny, base, small, medium, large-v3 | Больше = точнее, но медленнее |
| Язык | ru, en, auto | ru — русский + англ. термины |

### Скорость моделей

| Модель | Размер | GPU | CPU | Качество |
|--------|--------|-----|-----|----------|
| tiny | ~75 MB | <1 сек | 2-3 сек | Низкое |
| base | ~150 MB | 1-2 сек | 4-6 сек | Среднее |
| small | ~500 MB | 2-3 сек | 8-15 сек | Хорошее |
| **medium** | **~1.5 GB** | **3-5 сек** | **20-40 сек** | **Отличное** ★ |
| large-v3 | ~3 GB | 5-10 сек | 60+ сек | Максимальное |

**Рекомендация для русского языка: medium.**

Модели скачиваются автоматически. Можно удалить в настройках.

### Обработка текста

| Параметр | Описание |
|----------|----------|
| Слова-паразиты | Удаляет «ну», «вот», «типа», «как бы», «короче» |
| Голосовые команды | «Точка» → `.`, «запятая» → `,`, «новая строка» → перенос |
| Автопрослушивание | Запись без хоткея — по голосу |
| Автозапуск | Запускать с Windows |

## Транскрибация файлов

Правый клик по индикатору — Файлы.

1. Добавьте аудиофайлы (mp3, wav, m4a, ogg, flac)
2. Можно добавить несколько — обработаются по очереди
3. Включите «Определять говорящих» для диалогов
4. Экспорт:
   - **.txt** — текст с таймкодами
   - **.srt** — субтитры для видеоредакторов
   - **.vtt** — веб-субтитры для YouTube

---

## Дополнительные файлы

| Файл | Назначение |
|------|------------|
| `settings.json` | Все настройки (создаётся автоматически) |
| `history.json` | Последние 20 распознаваний |
| `replacements.json` | Словарь замен (создайте вручную) |
| `vocabulary.txt` | Термины для повышения точности |

### Словарь замен

Создайте `replacements.json`:
```json
{
  "реакт": "React",
  "питон": "Python",
  "нода": "Node.js",
  "гит": "Git"
}
```

### Vocabulary Boosting

Создайте `vocabulary.txt` — по одному термину на строку:
```
React
Python
Docker
Kubernetes
```

Whisper будет «ожидать» эти слова — точность повышается.

## FAQ

**Текст не вставляется?**
Убедитесь что курсор стоит в текстовом поле. Приложение использует Ctrl+V.

**Модель долго загружается?**
Первый раз — скачивается. Дальше из кэша — мгновенно.

**Whisper выдаёт мусор на тишине?**
Встроенный фильтр галлюцинаций отсекает «редактор субтитров» и подобное.

**Как ускорить?**
Установите PyTorch с CUDA → включите GPU в настройках.

**Английские слова транслитерируются?**
Язык `ru` (не `auto`) + добавьте термины в `vocabulary.txt`.
