# Speech-to-Text

Windows-приложение для голосового ввода. Удерживаете кнопку или хоткей — говорите — текст вставляется туда, где стоит курсор.

## Возможности

- **Голосовой ввод** — hold-to-record, текст вставляется в любое приложение
- **Глобальный хоткей** — запись из любого приложения (по умолчанию `ctrl+space`)
- **Стриминг** — текст появляется в реальном времени пока говорите
- **Два движка** — Whisper и Faster-Whisper (в 2-4 раза быстрее)
- **Русский + английские термины** — "используем React с threading" работает корректно
- **Голосовые команды** — "точка", "запятая", "новая строка" в режиме стриминга
- **Пост-обработка** — удаление слов-паразитов, словарь замен, авто-заглавные
- **Транскрибация файлов** — mp3, wav, m4a, ogg, flac с batch-очередью
- **Speaker diarization** — определение говорящих через whisperX
- **Экспорт** — .txt, .srt, .vtt (субтитры)
- **История** — последние 50 распознаваний с копированием
- **GPU-ускорение** — NVIDIA CUDA, авто-fallback на CPU
- **Локальная обработка** — ничего не отправляется в интернет

## Установка

### 1. Python зависимости

```bash
pip install -r requirements.txt
```

### 2. PyTorch с CUDA (для GPU)

```bash
pip install torch --index-url https://download.pytorch.org/whl/cu121
```

### 3. ffmpeg (для транскрибации файлов)

Скачайте с [ffmpeg.org](https://ffmpeg.org/download.html) и добавьте в PATH.

### 4. Hugging Face токен (для diarization, опционально)

Получите на [huggingface.co/settings/tokens](https://huggingface.co/settings/tokens) и укажите в настройках.

## Запуск

```bash
# GUI режим (по умолчанию)
python app.py

# Консольный режим
python app.py --console
```

При первом запуске откроется мастер настройки — выбор качества, загрузка модели, тест микрофона.

## Использование

### Голосовой ввод
1. Откройте приложение, где хотите ввести текст (Word, браузер, мессенджер)
2. Нажмите и удерживайте `ctrl+space` (или настроенный хоткей)
3. Говорите
4. Отпустите — текст вставится в поле ввода

### Транскрибация файлов
1. Правый клик по индикатору → "Транскрибация файлов"
2. Добавьте файлы (можно несколько)
3. Включите "Определять говорящих" при необходимости
4. Нажмите "Начать"
5. Экспортируйте результат в .txt, .srt или .vtt

## Настройки

Доступны через кнопку ⚙ в overlay или правый клик по индикатору → "Настройки".

| Настройка | Описание | По умолчанию |
|-----------|----------|--------------|
| Движок | Whisper / Faster-Whisper | faster-whisper |
| Модель | tiny / base / small / medium / large-v3 | base |
| Язык | ru / en / auto | ru |
| Хоткей | Глобальная горячая клавиша | ctrl+space |
| Слова-паразиты | Удалять "ну", "вот", "типа" | Вкл |
| Голосовые команды | "точка", "запятая" в стриминге | Выкл |

## Файлы

| Файл | Назначение |
|------|------------|
| `settings.json` | Настройки приложения |
| `replacements.json` | Словарь замен ("реакт" → "React") |
| `vocabulary.txt` | Термины для повышения точности |
| `history.json` | История распознаваний |

## Vocabulary Boosting

Создайте `vocabulary.txt` в папке проекта — по одному термину на строку:

```
React
Python
Docker
Kubernetes
API
```

Whisper будет "ожидать" эти слова, что повышает точность распознавания.
