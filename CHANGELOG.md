# Changelog

## [1.9.0] — 2026-03-18

### v1.9 — Стабильный релиз после 18 итераций

---

### v0.1 — MVP (консоль)
- Базовая структура проекта (core/, gui/, tests/)
- GPU-детектор (CUDA / CPU)
- Менеджер настроек (settings.json)
- Запись с микрофона (sounddevice)
- Распознавание через Faster-Whisper
- Консольный режим: Enter → запись → Enter → текст

### v0.2 — GUI overlay
- Overlay окно поверх всех окон (tkinter, тёмная тема)
- Hold-to-record кнопка
- Отслеживание активного окна (win32api)
- Вставка текста через Ctrl+V в целевое окно
- State machine (IDLE → RECORDING → PROCESSING → INSERTING)

### v0.3 — Настройки + движки
- Окно настроек с dropdown-ами
- Переключение Whisper / Faster-Whisper
- Менеджер моделей (lazy loading, кэш)
- Перезагрузка модели при смене настроек

### v0.4 — Глобальный хоткей + индикатор
- Глобальный хоткей через библиотеку keyboard
- Настраиваемая комбинация клавиш
- Плавающий индикатор (перетаскиваемый)
- Системный трей с меню

### v0.5 — Пост-обработка + история
- Удаление слов-паразитов (ну, вот, типа, как бы)
- Пользовательский словарь замен (replacements.json)
- Vocabulary boosting (vocabulary.txt)
- История распознаваний (20 записей)
- Окно истории (клик = копировать, двойной = вставить)

### v0.6 — Транскрибация файлов
- Импорт аудиофайлов (mp3, wav, m4a, ogg, flac)
- Пакетная обработка (очередь)
- Speaker diarization (определение говорящих)
- Экспорт: .txt, .srt, .vtt
- Окно результатов с таймкодами

### v0.7 — Стриминг + голосовые команды
- Распознавание в реальном времени (partial каждые 2.5 сек)
- Голосовые команды: «точка», «запятая», «новая строка»
- Отображение стриминг-текста в overlay

### v0.8 — Мастер первого запуска + UX
- First-run wizard (выбор движка, модели, тест микрофона)
- Тултипы (?) с подсказками в настройках
- Прогресс скачивания модели
- Звуковые уведомления (опционально)

### v0.9 — Полировка интерфейса
- Убран overlay — только компактный индикатор
- Фикс двойного нажатия хоткея
- Фикс вставки текста (keyboard.press_and_release вместо pyautogui)
- Фильтрация дубликатов микрофонов
- Одно окно настроек/истории/файлов (не множится)

### v1.0 — Автопрослушивание + GPU
- VAD (Voice Activity Detection) — запись без хоткея
- Установка PyTorch с CUDA (RTX 3060)
- Чекбокс GPU (отключается если нет CUDA)
- Статус установки движков (✓ / ✗)
- Статус моделей ([скачана] / [скачать ~500 MB])

### v1.1 — Фильтр галлюцинаций
- Фильтр «редактор субтитров», «Егорова» и т.д.
- Проверка RMS < 0.012 — тишина не отправляется в Whisper
- Минимум 0.8 сек записи
- Порог VAD увеличен

### v1.2 — Кнопки мыши + предупреждения
- Поддержка кнопок мыши как хоткей (средняя, боковые)
- Предупреждение при выборе неустановленного движка
- Неустановленные движки скрыты из списка
- Прогресс скачивания: поллинг размера файлов каждые 300ms

### v1.3 — Single instance + язык
- Защита от запуска нескольких копий (file lock)
- Язык по умолчанию: ru
- Фикс краша при сохранении настроек (main thread)

### v1.4 — Вставка текста v3
- keyboard.press_and_release("ctrl+v") — работает с хуками
- Win32 clipboard API (SetClipboardData)
- Сохранение и восстановление буфера обмена

### v1.5 — Стриминг в реальное поле
- Текст вставляется прямо в целевое приложение по мере произнесения
- Умная вставка: дописывает новые слова, не стирает старые
- Сравнение по словам (не символам) для минимизации стирания

### v1.6 — Спектрограф
- 7 вертикальных полосок вместо 3 точек
- Каждая на своей «частоте»
- Реакция на громкость голоса
- Прозрачный фон (только овал виден)

### v1.7 — HiDPI + дизайн настроек
- SetProcessDpiAwareness(2) — чёткие шрифты на 4K
- Компактные настройки без скролла
- Тёмная тема с карточками

### v1.8 — Управление моделями + автозапуск
- Удаление моделей из настроек
- Автозапуск с Windows (Startup folder)
- Рекомендация: medium для русского языка

### v1.9 — Документация + GitHub
- Подробный README с инструкциями
- MIT лицензия
- Публикация на GitHub
