# Changelog

## [1.0.0] — 2026-03-18

### Первый релиз

#### Голосовой ввод
- Глобальный хоткей (клавиатура или кнопка мыши) — зажал, сказал, отпустил
- Стриминг в реальном времени — текст появляется в поле по мере произнесения
- Умная вставка — дописывает новые слова, не стирает старые
- Сохранение буфера обмена — после вставки буфер восстанавливается

#### Распознавание
- Движки: Faster-Whisper (быстрый), OpenAI Whisper (оригинальный)
- Модели: tiny, base, small, medium, large-v3
- GPU-ускорение (NVIDIA CUDA) — в 5-10 раз быстрее CPU
- Русский язык с поддержкой английских терминов латиницей

#### Обработка текста
- Удаление слов-паразитов (ну, вот, типа, как бы)
- Пользовательский словарь замен (replacements.json)
- Vocabulary boosting (vocabulary.txt) — подсказки для модели
- Голосовые команды: «точка», «запятая», «новая строка»

#### Транскрибация файлов
- Поддержка: mp3, wav, m4a, ogg, flac
- Пакетная обработка (очередь файлов)
- Экспорт: .txt, .srt (субтитры), .vtt (веб-субтитры)
- Speaker diarization (определение говорящих) через whisperX

#### Интерфейс
- Компактный индикатор-спектрограф с аудио-реактивными полосками
- Прозрачный овальный фон (без прямоугольника)
- Настройки: тёмная тема, подсказки (?), управление моделями
- Системный трей
- HiDPI-поддержка (чёткие шрифты на 4K мониторах)

#### Дополнительно
- Автопрослушивание (VAD) — запись без нажатия хоткея
- Автозапуск с Windows
- Защита от запуска нескольких копий
- Фильтр галлюцинаций Whisper на тишине
- История: последние 20 распознаваний
