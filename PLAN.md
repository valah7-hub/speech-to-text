# Speech-to-Text — Требования и решения

## Требования

1. Windows-приложение с GUI (окно поверх всех окон)
2. Кнопка: удерживаешь — записывает с микрофона, отпускаешь — распознаёт
3. Результат вставляется туда, где стоит курсор (Word, браузер, любое приложение)
4. Настройки в интерфейсе:
   - Движок: Whisper / Faster-Whisper
   - Размер модели: tiny / base / small / medium / large-v3
   - Язык: ru / en / auto
5. GPU-ускорение (NVIDIA)

## Решения

- **Язык**: Python (уже установлен)
- **GUI**: tkinter (встроен в Python, ничего ставить не надо)
- **Запись звука**: sounddevice + numpy
- **Распознавание**: openai-whisper и faster-whisper (оба ставятся через pip)
- **GPU**: CUDA через PyTorch (для whisper) / CTranslate2 (для faster-whisper)
- **Вставка текста**: копируем в буфер обмена → имитируем Ctrl+V в предыдущее активное окно
- **Отслеживание окна**: win32api — запоминаем какое окно было активно до нажатия кнопки
- **Настройки**: сохраняются в settings.json рядом с приложением

## Зависимости (pip install)

- `openai-whisper` — оригинальный движок
- `faster-whisper` — ускоренный движок
- `sounddevice` — запись с микрофона
- `numpy` — обработка аудио
- `pyperclip` — работа с буфером обмена
- `pyautogui` — имитация нажатий клавиш
- `pywin32` — отслеживание активного окна Windows
